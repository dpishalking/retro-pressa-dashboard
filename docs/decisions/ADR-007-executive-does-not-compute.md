# ADR-007 — Why Executive does not compute from raw

- **Status:** Accepted (design intent / Standard)
- **Context:** Executive screens that recompute from Bitrix/SVOD create competing truths.
- **Decision:** Executive OS (planned) **reads** Management / Prediction / Export contracts only — does not invent formulas from raw warehouse.
- **Consequences:** Child OS own calculations; Executive is presentation + attention.
- **Open:** Exact Executive workbook — **Requires clarification** (not built yet).
